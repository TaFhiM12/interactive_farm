# Interactive Urban Farming Platform

## Run Project
1. `npm install`
2. Update `.env`
3. `npm run prisma:generate`
4. `npm run prisma:migrate`
5. `npm run seed`
6. `npm run dev`

## Open
- API: `http://localhost:5001/api/v1/health`
- Swagger: `http://localhost:5001/docs`
